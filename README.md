# lsdmatcher

Fuzzy matcer using Levenshtein String Distance

## Usage

```
Usage of ./lsdmatcher INFILE [TARGET]:
  -d int
    	distance (default 2)
```

## Installation

```
$ go get github.com/sfus/lsdmatcher
```
